# Recherche d'emploi
Application Web permettant d'avoir' les dernières offres d'emplois.

## Exigence
- Python >= 3.8
    - BeautifulSoup4 >= 4.9.*
    - Requests >= 2.24.*
    - Django == 3.1

## Utilisation
Simple d'utilisation, cette application web va vous permettre de ne pas rater une annonce dans vos recherches.
